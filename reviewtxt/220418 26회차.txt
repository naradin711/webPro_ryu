1.	Sequence
(1) 시퀀스의 정의 : 순차적인 번호를 자동으로 생성하는 객체로 테이블과 독립적으로 생성 및 저장 가능
(2) 특징 : 시퀀스에서 생성되는 번호는 유일하기 때문에 기본 테이블에서 인조 Primary Key 생성시 주로 사용
          여러 테이블에 의해 공유도 가능
          시퀀스는 테이블과 관계없이 생성, 저장. 오라클 내부 루틴에 의해 발생되고 증가, 감소된다.
(3)         문법 : 
 CREATE SEQUENCE FRIEND_SEQ -- 해당 시퀀스 이름 생성
  START WITH 1        -- 해당 숫자부터 시작한다. (기본값 1 )
  INCREMENT BY 1    -- 시퀀스가 반복될때마다 해당 숫자만큼 증가한다.
  MAXVALUE 9999     -- 최대값 지정
  MINVALUE 1          -- 최소값 지정
  NOCACHE              -- 새로운 시퀀스를 시작할때 캐시 메모리를 초기화 하는 행.
  NOCYCLE   ;           -- 시퀀스가 최대값을 넘으면 1이 되지 않게 하는 행.
